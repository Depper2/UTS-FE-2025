Medak Api

Asal: Lombok

Kategori: Adat Kelahiran

Latar Belakang: Medak Api merupakan tradisi
masyarakat Sasak di Lombok yang dilakukan sebagai
ritual pemberian nama bayi sekaligus doa keselamatan.

Penjelasan Adat Medak Api

Dalam tradisi Medak Api, keluarga biasanya mengundang pemuka adat atau tokoh agama untuk memimpin prosesi yang dilaksanakan beberapa hari setelah kelahiran bayi. Upacara ini ditandai dengan pembacaan doa, pemberian nama secara resmi, serta simbolisasi penggunaan api sebagai lambang penerangan hidup, penolak bala, dan harapan agar bayi tumbuh dalam jalan kebaikan.

Selain menjadi sarana spiritual, Medak Api juga memperkuat ikatan sosial karena melibatkan partisipasi keluarga besar dan masyarakat sekitar. Kehadiran mereka dalam upacara menegaskan pentingnya kebersamaan, gotong royong, dan peran komunitas dalam menyambut hadirnya seorang anggota baru dalam kehidupan sosial budaya Sasak.
